# ☕ Cafézão - Landing Page de Cafeteria

## 📌 Sobre o projeto

O **Cafézão** é uma landing page simples desenvolvida com o objetivo de praticar e aprimorar minhas habilidades em desenvolvimento Front-End.

A ideia desse projeto foi criar uma página moderna para uma cafeteria fictícia, utilizando conceitos de estruturação HTML, estilização com CSS e boas práticas de organização de código.

Foi um projeto desenvolvido em pouco tempo como um desafio pessoal para continuar praticando, testar novas ideias e manter meu GitHub ativo com projetos e evolução constante.

---

## 🚀 Funcionalidades

- 🎥 Vídeo de fundo na página inicial
- ☕ Layout temático de cafeteria
- 📌 Menu de navegação por seções
- 📖 Área "Sobre Nós"
- 🗺️ Integração com Google Maps usando iframe
- 🎨 Design moderno com cores personalizadas
- ✨ Efeitos visuais utilizando CSS

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JS

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto pratiquei:

- Estruturação semântica com HTML
- Organização de arquivos
- Uso de variáveis CSS (`:root`)
- Posicionamento de elementos com Flexbox
- Manipulação de vídeos como background
- Trabalhar com iframes externos
- Criação de layouts responsivos
- Melhoria na organização e limpeza do código

---

## 💡 Objetivo

Este projeto não tem como foco ser uma aplicação completa, mas sim servir como prática de desenvolvimento, experimentação de ideias e evolução contínua como desenvolvedor.

Pequenos projetos também fazem parte do processo de aprendizado e ajudam a fortalecer fundamentos importantes.
